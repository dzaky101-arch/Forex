EA License Generator

Buat kunci lisensi angka sederhana (Numeric Key) untuk membatasi akses Expert Advisor MT5 Anda berdasarkan Akun, Broker, atau Tanggal.

Parameter Lisensi

Konfigurasikan pembatasan untuk lisensi EA ini

Harus sama dengan SECRET_SALT yang tertulis di kode EA (.mq5) Anda

Kunci Lisensi Dihasilkan

Salin kunci angka ini dan masukkan ke parameter input EA di MT5

Cara Penggunaan

  1. Atur batasan lisensi di form sebelah kiri, lalu klik Generate License Key.
  2. Klik Salin Kunci untuk menyalin kode lisensi berupa deretan angka yang dihasilkan.
  3. Buka MetaTrader 5, pasang EA ke chart.
  4. Di tab Inputs pada pengaturan EA, cari parameter LicenseKey.
  5. Tempel (Paste) kunci lisensi angka tersebut ke dalam kolom nilai LicenseKey.
  6. Klik OK. EA akan langsung memvalidasi kunci tersebut secara otomatis.
